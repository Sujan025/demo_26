🚗 Smart Parking System

The Smart Parking System is an embedded project developed using Arduino UNO, IR sensors, LCD display, and LED . It is designed to automatically detect parking slot availability in real time and provide clear status updates to users. The system helps reduce the time required to find parking space and improves overall parking efficiency.

The system continuously monitors each parking slot using IR sensors. When a vehicle enters or leaves a slot, the sensor detects the change and sends the signal to the Arduino UNO. The microcontroller processes this data and instantly updates the status on the LCD display. At the same time, LED indicators provide a simple visual indication of whether a slot is occupied or available.

The main objective of this project is to develop a low-cost and efficient parking management system that can be implemented in places such as malls, offices, and public parking areas. It helps reduce traffic congestion caused by searching for parking spaces and improves user convenience through automation.

Overall, this project demonstrates how embedded systems can be used to solve real-world problems using simple sensors and microcontroller-based automation. 🚀
